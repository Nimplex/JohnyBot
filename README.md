# NudaBot
NudaBot, multi purpose discord bot.

## Requirments:
```
discord.js ^12.3.1
```

## Guide:

1. Install all dependencies
```shell
npm install
```

2. Create config.js file - it should look something like this
```js
module.exports = {
  prefix: ',',
  token: 'bot-token'
}
```

3. To run bot use
```shell
node src/index.js
```
